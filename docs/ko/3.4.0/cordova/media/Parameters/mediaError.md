# mediaError

미디어 기능에 오류가 있을 때 실행 되는 사용자 지정 된 콜백 함수.

    function(error) {
        // Handle the error
    }
    

## 매개 변수

*   **오류**: 장치에 의해 반환 된 오류. *(MediaError)*
