# compassOptions

나침반의 검색을 사용자 지정 하는 선택적 매개 변수.

## 옵션

*   **주파수**: 자주 밀리초에서 나침반 머리글을 검색 하는 방법. *(수)* (기본값: 100)

*   **필터**:도 watchHeading 성공 콜백을 시작 하는 데 필요한 변경. *(수)*

안 드 로이드 단점

---

*   `filter`지원 되지 않습니다.

*   `filter`지원 되지 않습니다.

## Windows Phone 7, 8 특수

*   `filter`지원 되지 않습니다.
