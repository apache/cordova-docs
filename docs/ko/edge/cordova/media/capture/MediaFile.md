# 돌아가기

> 미디어 캡처 파일의 속성을 캡슐화합니다.

## 속성

*   **이름**: 경로 정보 없이 파일 이름. (DOMString)

*   **fullPath**: 이름을 포함 한 파일의 전체 경로. (DOMString)

*   **유형**: 파일의 mime 형식 (DOMString)

*   **lastModifiedDate**: 날짜 및 시간 파일을 마지막으로 수정한. (날짜)

*   **크기**: 바이트에서 파일의 크기. (수)

## 메서드

*   **MediaFile.getFormatData**: 미디어 파일의 형식 정보를 검색 합니다.
