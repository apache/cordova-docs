# 嵌入 WebViews

科尔多瓦的应用程序通常是作为基于浏览器的*web 视图*在本机的移动平台内实现的。 这一节演示如何，为支持平台，以创建您自己的 web 视图组件，充分使用科尔多瓦的 api。 然后，您可以部署在混合应用程序中的本机组件以及这些科尔多瓦应用程序组件。

若要部署 web 视图，您需要熟悉每个本机编程环境。下面提供了支持的平台的说明：

*   亚马逊火 OS WebViews
*   Android WebViews
*   WebViews iOS
