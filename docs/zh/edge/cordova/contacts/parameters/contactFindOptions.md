# contactFindOptions

可选的参数， `contacts.find` 方法，用于筛选从联系人数据库返回的联系人。

    {筛选器：""、 多个： true，} ；
    

## 选项

*   **筛选器**： 用于过滤联系人的搜索字符串。*() DOMString*（默认值：`""`)

*   **多个**： 确定是否查找操作返回多个联系人。*（布尔）*（默认值：`false`)
