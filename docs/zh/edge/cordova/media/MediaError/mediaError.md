# MediaError

A `MediaError` 对象返回到 `mediaError` 时出现错误的回调函数。

## 属性

*   **代码**： 下面列出的预定义的错误代码之一。

*   **消息**： 错误消息描述该错误的详细信息。

## 常量

*   `MediaError.MEDIA_ERR_ABORTED`
*   `MediaError.MEDIA_ERR_NETWORK`
*   `MediaError.MEDIA_ERR_DECODE`
*   `MediaError.MEDIA_ERR_NONE_SUPPORTED`

## 说明

`MediaError`对象传递给 `mediaError` 回调函数时出现错误。
