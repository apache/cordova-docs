# CompassError

A `CompassError` 物件返回到 `compassError` 時出現錯誤的回呼函數。

## 屬性

*   **代碼**： 下面列出的預定義的錯誤代碼之一。

## 常量

*   `CompassError.COMPASS_INTERNAL_ERR`
*   `CompassError.COMPASS_NOT_SUPPORTED`

## 說明

當錯誤發生時， `CompassError` 物件作為一個參數傳遞 `compassError` 回呼函數。
