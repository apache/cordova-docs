1

Installing Cordova

Cordova command-line runs on Node.js and available on NPM. Follow platform specific guides to install additional platform dependencies. Open a command prompt or Terminal, and type npm install -g cordova

$ npm install -g cordova
2

Create a project

Create a Cordova project using the command-line tool to create a blank project. Navigate to the directory where to create your project and type cordova create <PATH>

For a complete set of options, type cordova help create

$ cordova create MyApp
3

Add a platform

After creating a Cordova project, navigate to the project directory. From the project directory, you need to add a platform you want to build your app.

To add platform, type cordova platform add <platform name>.

For a complete list of platforms you can add, run cordova platform

$ cd MyApp

$ cordova platform add browser

4

Run your app

From the command line, run cordova run <platform name>.

You can replace browser with the platform that you would like to run (e.g. iOS, Android, Windows)

$ cordova run browser

Supported Platforms

See a list of features supported in each platform

Cordova wraps your HTML/JavaScript app into a native container which can access the device functions of several platforms. These functions are exposed via a unified JavaScript API, allowing you to easily write one set of code to target nearly every phone or tablet on the market today and publish to their app stores.
