Want to Contribute?

Contributors are needed to keep the project moving forward!

To share your contributions with the community, you can send a pull request on GitHub to the Apache git mirrors. You can also advocate for your changes directly on our dev mailing list.

Get Started Now

  1. Join the Mailing List
  2. Sign the Individual Contributor License Agreement (ICLA)
  3. Create an account in Apache Jira
  4. Bookmark the official URL for Cordova Issues (See also Issue Workflow)
  5. Send a brief introduction of yourself to dev@cordova.apache.org
