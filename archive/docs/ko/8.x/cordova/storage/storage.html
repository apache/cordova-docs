스토리지

코르 도우 바에 대 한 저장소 옵션에 대 한 개요입니다.

여러 저장소 Api 코르도바 애플리케이션에 사용할 수 있습니다. Html5rocks를 참조 하십시오. 더 완전 한 개요 및 예제.

LocalStorage

또한 다른 세션 저장소 인터페이스 또는 웹 스토리지, 간단한 스토리지,으로 알려진,이 API 동기 키/값 쌍 스토리지를 제공 하 고 기본 WebView 구현에서 사용할 수 있습니다. 자세한 내용은 W3C 사양을 참조 하십시오.

WebSQL

이 API는 기본 WebView에서 사용할 수 있습니다. 웹 SQL 데이터베이스 사양 더 완전 한 기능의 데이터베이스 테이블 액세스 SQL 쿼리를 통해 제공 합니다.

다음 플랫폼 WebSQL을 지원합니다.

  • 안 드 로이드
  • 블랙베리 10
  • iOS
  • Tizen

IndexedDB

이 API는 기본 WebView에서 사용할 수 있습니다. 색인 DB LocalStorage 보다 더 많은 기능이 있지만 WebSQL 보다 적게 제공합니다.

다음 플랫폼 IndexedDB를 지원합니다.

  • 블랙베리 10
  • Firefox 운영 체제
  • Windows Phone 8
  • 윈도우 8

플러그인 기반 옵션

위에 나열 된 Api 저장소 뿐만 아니라 파일 API 로컬 파일 시스템에 캐시 데이터를 수 있습니다. 코르 도우 바 플러그인 의 다른 유사한 스토리지 옵션을 제공 합니다.
