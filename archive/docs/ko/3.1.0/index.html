가이드

  • 개요

    Cordova가 처음이라면?

  • 커맨드라인 인터페이스

    커맨드라인으로 생성, 빌드, 그리고 배치까지

  • 플랫폼 가이드

    해당하는 SDK설치와 프로젝트 업데이트

  • Config.xml 파일

    응용 프로그램의 기능을 사용자 지정 합니다.

  • 아이콘 및 시작 화면

    애플 리 케이 션의 표시 이미지를 사용자 지정 합니다.

  • WebViews 포함

    프로젝트에서 코르도바 WebView를 구현 합니다.

  • 플러그인 개발 가이드

    첫 번째 플러그인을 개발 합니다.

  • 개인 정보 보호 가이드

    중요 한 모바일 개인 정보 보호 문제에 대해 자세히 알아보기

  • 화이트 리스트 가이드

    외부 리소스에 대 한 응용 프로그램 액세스를 부여 합니다.

API 참조

  • 가속도센서

    기기의 모션센서 활용하기

  • 카메라

    기기의 카메라를 이용하여 사진 찍기

  • 캡쳐

    기기의 미디어 캡쳐 어플리케이션을 이용하여 미디어 파일 캡쳐하기

  • 나침반

    기기가 가리키는 방향 얻기

  • 네트워크 연결

    네트워크 상태와 셀룰러 네트워크 정보 빠르게 확인하기

  • 연락처

    기기의 연락처 데이터베이스 사용하기

  • 장치

    기기의 특정 정보 수집하기

  • 이벤트

    자바스크립트를 통해 내장 이벤트에 연결하기

  • 파일

    자바스크립트를 통해 내장 파일 시스템에 연결하기

  • 위치정보

    어플리케이션에서 위치정보 이용하기

  • Globalization

    지역에 따라 객체 표현을 다르게 하기

  • 인앱브라우저

    다른 인앱브라우저 객체에서 URL 실행하기

  • 음악

    오디오 파일 기록 및 재생하기

  • 알림

    기기의 시각, 청각과 촉각 알림

  • 스플래시 스크린

    어플리케이션의 스플래시 스크린 보이기 및 숨기기

  • 스토리지

    기기의 내장 저장소 옵션으로 연결하기
