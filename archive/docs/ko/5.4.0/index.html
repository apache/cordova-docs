가이드

  • 개요

    코르 도우 바에 새로운 경우 여기를 시작 합니다. 설치 및 다음 단계를 포함 한다.

  • 플랫폼 지원

    모든 주요 기능에 대 한 호환성 테이블입니다.

  • 명령줄 인터페이스

    만들기, 빌드 및 명령줄에서 배포.

  • 플랫폼 가이드

    각 플랫폼 SDK 설정 하 고 프로젝트를 업데이트 합니다.

  • Plugman를 사용 하 여 플러그인을 관리 하

    플랫폼을 중심으로 워크플로 사용 하는 경우 CLI 없이 플러그인을 관리 합니다.

  • Config.xml 파일

    응용 프로그램의 기능을 사용자 지정 합니다.

  • 아이콘 및 시작 화면

    응용 프로그램의 표시 이미지를 사용자 지정 합니다.

  • WebViews 포함

    네이티브 프로젝트에서 코르도바 WebView를 구현 합니다.

  • 플러그인 개발 가이드

    자신의 플러그인을 개발 합니다.

  • 개인정보보호 가이드

    모바일 개인정보보호 이슈에 대해 알아보기

  • 보안 가이드

    정보 및 보안 응용 프로그램을 구축 하기 위한 팁입니다.

  • 플랫폼 및 플러그인 버전 관리

    저장 하 고 번거로움 없이 알려진 상태로 CLI 프로젝트 복원.

  • 화이트 리스트 가이드

    외부 리소스는 응용 프로그램 액세스를 부여할.

  • 스토리지

    기본 저장소 옵션에 대 한 개요.

  • 후크 가이드

    사용자 지정 스크립트를 추가 하 여 기본 코르도바 기능을 확장.

  • 다음 단계

    새로운 코르도바 개발자가 발생 하는 항목을 봐.
